# serra
For beauty style

## usage
```html
<head>
...
  <link rel="stylesheet" href="./path_to_serra/dist/serra.min.css" />
...
</head>
<body>
...
  <div class="sr-box wrap">
    <div class="grow-1"></div>
    <div class="grow-2"></div>
  </div>
...
</body>
```

## develop
```shell
npm run dev # develop
npm run build # build
```
